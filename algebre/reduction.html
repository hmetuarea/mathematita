Réduction des endomorphismes

Description

Ce cours présente les bases de l'algèbre linéaire dans le cas de la dimension finie ainsi que les notions élémentaires relatifs à celles-ci pour résoudre divers problèmes mathématiques : équation différentielle, analyse fonctionnelle, etc.
